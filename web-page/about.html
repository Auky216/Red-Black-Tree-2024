¿Qué son los árboles Red-Black?

Los árboles Red-Black son un tipo de árbol binario de búsqueda autobalanceado. Cada nodo de un árbol Red-Black almacena un bit adicional que representa el color del nodo, ya sea rojo o negro. Este color se usa para asegurar que el árbol se mantenga balanceado durante las inserciones y eliminaciones.

  • Propiedad 1: Cada nodo es rojo o negro.
  • Propiedad 2: La raíz es negra.
  • Propiedad 3: Todas las hojas (NIL) son negras.
  • Propiedad 4: Los nodos rojos no pueden tener hijos rojos (no pueden haber dos nodos rojos consecutivos).
  • Propiedad 5: Todo camino simple desde un nodo a una hoja descendiente contiene el mismo número de nodos negros.
